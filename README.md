# 📄 DocuMaster

A modern, fast, and offline Flutter utility app for handling all your document needs — convert, compress, and manage files with ease.

---

## 🚀 Features

- 🖼️ **Image → PDF**
  - Convert multiple images into a single PDF
  - Customize page size, margins, and quality

- 📄 **PDF → Image**
  - Extract PDF pages as images
  - Export images individually (no ZIP)

- 🗜️ **Compress Image**
  - Reduce image size with minimal quality loss

- 📉 **Compress PDF**
  - Optimize PDF file size efficiently

- 🕘 **History**
  - View and manage previously converted files
  - Download, share, or delete anytime

---

## 🎯 Key Highlights

- ⚡ Fast and lightweight
- 🔒 Works completely offline (no login required)
- 🎨 Modern dark UI (Material 3 inspired)
- 📱 Clean and user-friendly experience
- 📂 Local storage support for file history

---

## 🛠️ Tech Stack

- **Flutter** (Latest Stable)
- **Dart**
- **State Management:** Provider / Riverpod
- **Local Storage:** Hive / Sqflite
- **File Handling & Processing:** Flutter plugins
